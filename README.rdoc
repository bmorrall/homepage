== Vagrant Development

This project uses {Vagrant}[http://vagrantup.com] for development, use `vagrant up` to setup a development virtual machine. Run `vagrant ssh` to access it.

=== Headerless Tests

The Vagrant setup script installs {Xvfb}[http://en.wikipedia.org/wiki/Xvfb] for headerless testing. Run the following commands to run tests in headerless mode.

    export DISPLAY=:99
    sudo /etc/init.d/xvfb start
    # Run headerless tests (i.e. bundle exec cucumber)
    sudo /etc/init.d/xvfb stop
